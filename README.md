# Ecological Memory and Hysteresis in Complex Ecosystems

## Description
Data and reproducible code for: "Ecological Memory Generates Scale-Dependent 
Hysteresis in Complex Ecosystems"

## Citation
Adeyemi, S. (2026). Ecological Memory Generates Scale-Dependent Hysteresis 
in Complex Ecosystems. *Proceedings of the National Academy of Sciences*. 
(In review)

## Repository Contents
- `code/` - R scripts for analysis and figure generation
- `data/` - Time series data for 72 regime-shifting and 12 stable sites
- `figures/` - Generated publication-quality figures
- `README.md` - This file

## Requirements
- R version 4.3.1 or higher
- Required packages: ggplot2, fracdiff, fractal, dplyr, tidyr, patchwork, etc.

## Usage
See individual R scripts for detailed instructions.
